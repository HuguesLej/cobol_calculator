# cobol_calculator

This project is not terminated yet.

A simple calculator coded in Cobol.
The use of `COMPUTE` statement is voluntarily avoided to perform operations.
It allows you to perform basic operations such as:
- `+` (add)
- `-` (subtract)
- `*` (multiply)
- `**` (exponent)
- `/` (divide)
- `%` (modulo)
